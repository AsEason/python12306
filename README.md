#### 准备工作
1. 安装浏览器驱动，这里是用的谷歌浏览器。**mac安装方式：brew install chromedriver**
2. 安装splinter，Splinter是一个自动化测试网络应用的Python库，可以将打开浏览器、输入URL、填写表单、点击按钮等全部操作自动化。**安装方式：pip install splinter**. 

#### 发送邮箱
